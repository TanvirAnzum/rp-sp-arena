# Rangpur Sports Arena — Who Can Do What

| Feature | Staff | Owner |
|---|:---:|:---:|
| **Turf Booking** | | |
| Book a slot for a customer | ✅ | ✅ |
| Cancel / reschedule a booking | ✅ | ✅ |
| Add / edit / delete time slots | ❌ | ✅ |
| Set or change turf prices | ❌ | ✅ |
| **Swimming** | | |
| Issue a swimming token | ✅ | ✅ |
| Mark token paid/unpaid | ✅ | ✅ |
| Set or change swimming price | ❌ | ✅ |
| **Food Sales** | | |
| Sell food items (existing list) | ✅ | ✅ |
| Add / edit / remove food items | ❌ | ✅ |
| Change food prices | ❌ | ✅ |
| **Other Items Sales** | | |
| Sell items (existing list) | ✅ | ✅ |
| Add / edit / remove items | ❌ | ✅ |
| Change item prices | ❌ | ✅ |
| **Dashboard & Reports** | | |
| View today's sales | ✅ | ✅ |
| View daily/monthly revenue summary | ❌ | ✅ |
| View revenue charts | ❌ | ✅ |

**In short:**
- **Staff** can take bookings, sell items, and issue tokens using whatever slots/prices/products are already set up.
- **Owner** controls all the setup — slot times, prices, product lists — and is the only one who sees full revenue reports.
